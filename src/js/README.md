# Архитектура клиентского кода

`assets/js/archive.js` — собранный файл для сайта. Его не нужно править руками.

Рабочие исходники живут здесь:

- `00-core/` — данные, справочники, конфигурация разделов, состояние и общие утилиты.
- `01-render/` — всё, что превращает данные в разметку: общие блоки, каталог, детальные страницы.
- `02-routing/` — меню, адреса, переходы и восстановление позиции каталога.
- `03-app/` — события, основной рендер, запуск, адаптивность и скроллбары.

После правок запускай:

```bash
python tools/build_js.py
python tools/check_archive.py
```
